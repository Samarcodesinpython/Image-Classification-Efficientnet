# 📂 Source Directory

This directory is reserved for modular Python scripts extracted from the notebook.

## Planned Modules

| Module          | Description                                          |
| --------------- | ---------------------------------------------------- |
| `dataset.py`    | Custom PyTorch Dataset class and data transforms     |
| `model.py`      | Model architecture setup and configuration           |
| `train.py`      | Training loop with validation and checkpointing      |
| `inference.py`  | Inference pipeline with test-time augmentation       |
| `utils.py`      | Helper functions (seed setting, metrics, etc.)       |

## Status

Currently, all logic resides in the Jupyter notebook at `notebooks/scene_classification.ipynb`. Modularization into this directory is planned as a future improvement.
